5.1Solving Basic Equations ▶ Part 1 ▶ Part 2 ▼

Connect to prior knowledge: An equation is a statement that two algebraic expressions are equal — think of it like a scale. Both sides are balanced. Whatever you do to one side you must do to the other. Always. A variable is just a number we don't know yet — don't let it freak you out!

Key concept — solution
A solution to an equation is a number that, when you plug it in for x, makes the equation true. That's the magic number we're solving for.

5 key principles for organizing your work

  1. Work vertically — top down. Do not work left to right. Do not erase.
  2. Line up your equal signs. Draw a vertical line through the equal sign to stay organized.
  3. Whatever you do to one side, you do to the other. Always. This keeps the equation balanced.
  4. Keep going until x is by itself. Circle it. That's the goal.
  5. Write “Check” and substitute your answer back in. Use ✓ if it works, ✗ if it doesn't.

The “Undo” table — your most important tool

Ask: what is happening to x? List it top-to-bottom following PEMDAS. Then undo in reverse order bottom-to-top.

↓ fill in top to bottomundo bottom to top ↑
What's happening to xHow to undo it
Adding a number  (x + 5)→Subtract from both sides
Subtracting a number  (x − 5)→Add to both sides

Gotcha — the invisible coefficient
When you see −x, there is a −1 right there — it's invisible. So −x is really −1·x. Watch your signs!

5.2Solving Equations with Fractions ▶ Watch ▼

Good news and bad news. Bad news: fractions show up in equations. Ugghh!!! Good news: we can use the exact same technique from 5.1. Same undo table — no new tools needed.

Case 1 — x divided by a number: solve x/4 + 3 = 8

Worked example — Case 1
x/4 + 3  =  8 ← x is ÷4, then +3. Undo: −3 first, then ×4        −3     −3      x/4  =  5        ×4     ×4        x  =  20  ✓ Check: 20/4 + 3 = 5 + 3 = 8 ✓

Case 2 — x multiplied by a fraction: solve (2/3)x = 10

Worked example — Case 2 — multiply by the reciprocal
(2/3)x  =  10 ← x is ×(2/3). Undo: multiply by the reciprocal 3/2 (3/2)·(2/3)x  =  10·(3/2) ← the 2s and 3s cancel         x  =  15  ✓ Check: (2/3)(15) = 10 ✓

Case 3 — compound fraction: solve (3x + 1)/4 = 5

Worked example — Case 3 — watch the implicit parentheses!
(3x + 1)/4  =  5 ← think: (3x + 1) ÷ 4 = 5. Undo ÷4 first              ×4     ×4        3x + 1  =  20              −1     −1             3x  =  19              ÷3     ÷3              x  =  19/3  ✓ Check: (3(19/3)+1)/4 = 20/4 = 5 ✓
Sign errors in numerators
When you see (6 − 3x)/5, think (6 − 3x) ÷ 5. The −3 stays with x. Say it out loud: “negative three times x.”
Joke: 4 out of 3 people struggle with fractions.
5.3Solving More Complex Equations ▶ Watch ▼

What if an equation has parentheses, multiple terms on each side, or x on both sides? Double underline that. We have a process — and we already know most of it.

Process for solving equations (4 steps)
Step 1 — Simplify each side completely:
  • Distribute to remove parentheses — circle negative signs first
  • Combine like terms on each side
  • Cover up the right side and deal with the left. Then switch.

Step 2 — Get x on one side. Move the x term with the smaller coefficient — fewer negatives means fewer mistakes.

Step 3 — Undo using the table from 5.1

Step 4 — Check your solution
Negative sign before parentheses
When you see −(2x + 7), write a 1: −1(2x + 7). Circle that −1. It multiplies every term inside. Say it out loud as you go.

Worked example — solve: 2(x + 4) = 3x − 1

Worked example
2(x + 4)  =  3x − 1 2x + 8  =  3x − 1 ← distribute −2x           −2x ← move smaller coefficient (2 < 3) 8  =  x − 1 +1              +1 9  =  x  ✓ Check: 2(9+4) = 26; 3(9)−1 = 26 ✓
Joke: “Dear Algebra, please stop asking me to find your x. She's not coming back and don't ask y.”
5.4Special Case Equations ▶ Watch ▼

Important: we solve equations. We simplify and evaluate algebraic expressions. An equation has an equal sign. An expression does not. You cannot solve an expression.

Common sense first — try numbers in: x + 1 = x + 2
x = 7:  7+1=8  vs  7+2=9  →  8≠9. No. x = 0:  0+1=1  vs  0+2=2  →  1≠2. No. How could any number +1 equal the same number +2? It just can't.
The rule — when x disappears in Step 2
FALSE statement (like 3 = 7) → No Solution
Would you rather have $1 or $2? Not equal. No value of x works.

TRUE statement (like 6 = 6) → All Real Numbers
Every number makes the equation true.

No Solution — solve: 2x + 3 = 2x + 9

Worked example
2x + 3  =  2x + 9 −2x           −2x 3  =  9  ← FALSE → No Solution

All Real Numbers — solve: 12x − 3(x − 2) = 9x + 6

Worked example
12x − 3(x − 2)  =  9x + 6 9x + 6  =  9x + 6 ← distribute, combine −9x         −9x 6  =  6  ← TRUE → All Real Numbers Check: pick any number — they all work.
Joke: “Just another thing I hate about algebra… when the solution is ‘no solution’. Ugghh!!!”

5.4
Solve vs. Simplify vs. Evaluate
Solve = find x in an equation. Simplify = reduce an expression. Evaluate = find the value of an expression for a given x.
Visual example
Solve: 2x−1=7 → x=4
Simplify: 3x+2x → 5x
Evaluate: 3x at x=2 → 6
5.4
Solve
To find the value(s) of the variable that make an equation true. You can only solve equations — not expressions.

→ 4x = 12
→ x = 3

When is the solution “No Solution”?
When x cancels in Step 2 and a FALSE statement remains (like 3 = 7). No value of x makes the equation true.
Visual example

−5x     −5x
—————

When is the solution “All Real Numbers”?
When x cancels in Step 2 and a TRUE statement remains (like 6 = 6). Every number makes the equation true.

−9x     −9x
—————
