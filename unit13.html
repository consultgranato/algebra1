13.1Introduction to Polynomials ▼

Connect to prior knowledge: You've been working with expressions that have variables. A polynomial is just a collection of terms with non-negative integer exponents — no square roots, no negative exponents, no variables in denominators.

Types of Polynomials
Monomial: 1 term   (e.g., 5x²)
Binomial: 2 terms   (e.g., 3x + 4)
Trinomial: 3 terms   (e.g., x² + 2x − 5)

Degree: The highest exponent in the polynomial
Standard form: Write terms in descending order of degree
Leading coefficient: Coefficient of the highest-degree term

Worked example — identify degree and leading coefficient

Worked example
Polynomial: 4x³ − 7x + 2 Degree: 3 (highest exponent is 3) Leading coefficient: 4 Standard form: already in order ✓

Worked example — write in standard form

Worked example
5 + 3x² − x → 3x² − x + 5 (descending order: degree 2, then 1, then 0)
Joke: Why can't you trust a polynomial? Because they always have terms.
13.2Adding & Subtracting Polynomials ▼

Connect to prior knowledge: Adding and subtracting polynomials is really just combining like terms. Remember: like terms have the same variable AND the same exponent.

Adding Polynomials
Identify like terms and combine. Like terms: same variable, same exponent.
x² and 3x² are like terms.   x² and x³ are NOT.
Subtracting Polynomials — DANGER ZONE
When subtracting, distribute the negative sign to EVERY term in the second polynomial. Use parentheses! Say it out loud as you go.

Worked example — adding

Worked example
(3x² + 2x − 1) + (x² − 5x + 4) = (3x² + x²) + (2x − 5x) + (−1 + 4) = 4x² − 3x + 3 ✓

Worked example — subtracting

Worked example — distribute the negative!
(5x² + 3x − 2) − (2x² − x + 1) = 5x² + 3x − 2 − 2x² + x − 1 ← distribute − = 3x² + 4x − 3 ✓
13.3Multiplying Polynomials (Monomial × Polynomial) ▼

Connect to prior knowledge: Use the distributive property to multiply a monomial by every term in the polynomial. When multiplying variables, add exponents (from Unit 11).

Rules for Monomial Multiplication
Coefficients: multiply together
Same-base variables: add the exponents
Example: 3x² × 4x³ = 12x⁵

Worked example — distribute

Worked example
3x(2x² − 5x + 4) = 3x(2x²) + 3x(−5x) + 3x(4) = 6x³ − 15x² + 12x ✓
Worked example 2
−2x²(3x − 7) = −2x²(3x) + (−2x²)(−7) = −6x³ + 14x² ✓
13.4Multiplying Polynomials (Binomial × Binomial — FOIL) ▼

Connect to prior knowledge: When multiplying two binomials, use FOIL to make sure you multiply every term by every other term.

FOIL = First + Outer + Inner + Last
(a + b)(c + d):
First: a × c
Outer: a × d
Inner: b × c
Last: b × d
Then combine like terms.

Worked example — (x + 3)(x + 5)

Worked example
F: x × x = x² O: x × 5 = 5x I: 3 × x = 3x L: 3 × 5 = 15 = x² + 5x + 3x + 15 = x² + 8x + 15 ✓

Worked example — (2x − 3)(x + 4)

Worked example
F: 2x × x = 2x² O: 2x × 4 = 8x I: −3 × x = −3x L: −3 × 4 = −12 = 2x² + 8x − 3x − 12 = 2x² + 5x − 12 ✓
Special Case: Difference of Squares
(a + b)(a − b) = a² − b²
Example: (x + 4)(x − 4) = x² − 16
Joke: What did the polynomial say to the monomial? You're just a part of a bigger picture.

Degree of a Polynomial
The highest exponent of the variable in the polynomial.
Example
4x³ − 2x + 1
Degree = 3 (highest exponent)
13.1
Leading Coefficient
The coefficient of the term with the highest degree (when written in standard form).
Example
7x² + 3x − 5
Leading coefficient = 7
13.1
Standard Form
Writing the polynomial with terms in descending order of degree (highest to lowest exponent).
Example
Not standard: 2 + x − 3x²
Standard: −3x² + x + 2

13.2
Like Terms
Terms that have the SAME variable raised to the SAME exponent. You can only add/subtract like terms.
Example
Like: 3x² and 7x²
Not like: 3x² and 7x
Not like: 3x² and 7x³

13.4
FOIL
First, Outer, Inner, Last — a method for multiplying two binomials by distributing every term in the first by every term in the second.
Example
(x+2)(x+3):
F: x², O: 3x, I: 2x, L: 6
= x² + 5x + 6
