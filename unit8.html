8.1
Slope
A measure of steepness of a line. Calculated as rise over run, or (y₂−y₁)/(x₂−x₁).
Example
Points (0,0) and (3,6):
m = (6−0)/(3−0) = 2
slope = 2
8.1
Rise / Run
Rise = vertical change (up or down). Run = horizontal change (left or right). Slope = rise ÷ run.
Example
From (1,2) to (4,8):
Rise = 8−2 = 6
Run = 4−1 = 3
m = 63 = 2

8.2
Slope-Intercept Form y = mx + b
Standard form for a linear equation. m is the slope; b is the y-intercept.
Example
y = 3x − 2
m = 3, b = −2
y-intercept at (0, −2)
8.2
m (slope)
The coefficient of x in y = mx + b. Tells you how much y changes for every 1 unit increase in x.
Example
y = 5x + 1
m = 5 → for every +1 in x, y increases 5
8.2
b (y-intercept)
The constant in y = mx + b. The y-value where the line crosses the y-axis (when x = 0).
Example
y = 2x + 7
b = 7 → line crosses y-axis at (0, 7)
8.2
x-intercept
Where the line crosses the x-axis. Set y = 0 and solve for x.
Example
y = 2x − 6
0 = 2x − 6 → x = 3
x-intercept: (3, 0)
New in 8.3
8.3
Point-Slope Form
y − y₁ = m(x − x₁). Used to write an equation when you know the slope and one point.
Example
m=2, point (3,5):
y − 5 = 2(x − 3)
→ y = 2x − 1
8.3
Parallel Lines
Two lines that never intersect. They have the same slope but different y-intercepts.
Example
y = 3x + 1
y = 3x − 5
Both have m = 3
8.3
Perpendicular Lines
Two lines that intersect at 90°. Their slopes are negative reciprocals of each other: m₂ = −1/m₁.
Example
m = 2 → perpendicular slope: −12
m = −3 → perpendicular slope: 13
New in 8.4
8.4
Standard Form Ax + By = C
Another form for a linear equation. A, B, C are integers; A must be non-negative. Useful for finding intercepts quickly.
Example
2x + 3y = 12
x-int: (6, 0)
y-int: (0, 4)
8.4
Horizontal Line y = k
A line where every point has the same y-value. Slope = 0.
Example
y = 5 → horizontal line
passes through (0,5), (3,5), (−2,5)
slope = 0
8.4
Vertical Line x = h
A line where every point has the same x-value. Slope is undefined.
Example
x = −3 → vertical line
passes through (−3,0), (−3,2), (−3,−1)
slope: undefined
