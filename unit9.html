9.1What is a System of Equations? ▼

Connect to prior knowledge: You already know how to check if an ordered pair is a solution to a single equation. Now we level up — a system of equations requires the ordered pair to satisfy BOTH equations at the same time.

Key concept — system of equations
A system of equations is a group of linear equations that depend on each other and must be solved together.

We solve systems of 2 linear equations with 2 variables (x and y).

To solve = find the ordered pair (x, y) that makes BOTH equations true simultaneously.
Mr. Behling says: Don't let it freak you out — it's just two equations working together.

Checking if an ordered pair is a solution

Process for checking a solution
Step 1: Substitute the x and y values into Equation 1. Does it work?
Step 2: Substitute the same x and y values into Equation 2. Does it work?
Both must be true for it to be a solution to the system.

Worked example 1 — Is (2, 3) a solution?

Check (2, 3) in y = x + 1 and x + y = 5
Equation 1: y = x + 1 → 3 = 2 + 1 → 3 = 3 ✓ Equation 2: x + y = 5 → 2 + 3 = 5 → 5 = 5 ✓ Both true → Yes, (2, 3) IS a solution.

Worked example 2 — Is (−1, 4) a solution?

Check (−1, 4) in y = −3x + 1 and 2x + y = 2
Equation 1: y = −3x + 1 → 4 = −3(−1) + 1 → 4 = 3 + 1 → 4 = 4 ✓ Equation 2: 2x + y = 2 → 2(−1) + 4 = 2 → −2 + 4 = 2 → 2 = 2 ✓ Both true → Yes, (−1, 4) IS a solution.
9.2Solving Systems by Graphing ▼

Connect to prior knowledge: Each equation in a system is a line. You already know how to graph lines using slope-intercept form. Now graph both lines and find where they intersect.

Key idea
The intersection point is the solution — it is the one point that satisfies BOTH equations at the same time.

Process for solving by graphing

  1. Graph both lines using slope-intercept form (y = mx + b)
  2. Identify the intersection point (where the lines cross)
  3. Write as ordered pair (x, y) — that is your answer
  4. Check: substitute back into both equations to verify

Three possibilities when graphing 2 lines

What can happen?
One solution: Lines cross at one point (different slopes) → intersection point is the answer

No solution: Lines are parallel — same slope, different y-intercept → they never meet

Infinite solutions: Lines are the same line (coincident) — same slope and same y-intercept → every point works

Worked example — solve by graphing

Solve: y = 2x − 1 and y = −x + 5
Step 1 — Identify slope & y-intercept for each line:   Line 1: y = 2x − 1 → m = 2, b = −1 → plot (0,−1), go up 2 right 1   Line 2: y = −x + 5 → m = −1, b = 5 → plot (0,5), go down 1 right 1 Step 2 — Find intersection:   Lines cross at (2, 3) (read from graph) Step 3 — Check in BOTH equations:   Eq 1: 3 = 2(2)−1 = 3 ✓    Eq 2: 3 = −2+5 = 3 ✓ Solution: (2, 3)
Graph: two lines intersecting at the solution
Note
Graphing gives approximate answers. Substitution and elimination give exact answers every time.
9.3Solving Systems by Substitution ▼

Connect to prior knowledge: Substitution is about replacing one variable with an equivalent expression. The goal: get one equation with just ONE variable, solve it, then find the other.

Goal of substitution
Get one variable alone in one equation, then substitute that expression into the other equation. You're left with one equation and one variable — solve it!

Process for solving by substitution

  1. Solve one equation for x or y (pick whichever is easiest)
  2. Substitute that expression into the OTHER equation
  3. Solve for the remaining variable
  4. Back-substitute to find the other variable
  5. Write answer as (x, y)
  6. Check both equations
Key tip
You can solve for either variable in either equation — pick whichever is easiest (already isolated, or coefficient of 1).

Worked example 1

Solve: y = 2x − 1 and x + y = 8
Step 1 — y is already alone: y = 2x − 1 Step 2 — Substitute into Eq 2:   x + (2x − 1) = 8 ← replace y with (2x−1)   3x − 1 = 8 → 3x = 9 → x = 3 Step 3 — Back-substitute: y = 2(3) − 1 = 5 Solution: (3, 5)   Check: 5=2(3)−1 ✓ and 3+5=8 ✓

Worked example 2

Solve: 3x + y = 7 and x − 2y = 0
From Eq 2: x = 2y Substitute into Eq 1: 3(2y) + y = 7                    6y + y = 7                        7y = 7 → y = 1 Back-sub: x = 2(1) = 2 Answer: (2, 1)
9.4Solving Systems by Elimination ▼

Connect to prior knowledge: Elimination is a power move — we add the two equations together to make one variable disappear. No substitution needed, just addition!

Goal of elimination
Add the two equations together to cancel (eliminate) one variable. The coefficients of that variable must be opposites (like +3y and −3y).
When to use each method
Graphing Best for visualizing; approximate answers

Substitution Best when one variable is already isolated (y=... or x=...)

Elimination Best when both equations are in standard form (ax+by=c)

Process for solving by elimination

  1. Line up the equations with like terms in columns
  2. Multiply one or both equations so that coefficients of one variable are opposites
  3. Add the equations together — one variable cancels
  4. Solve for the remaining variable
  5. Substitute back to find the other variable
  6. Check both equations
Tip
Pick the variable with the smaller coefficient to multiply — easier numbers to work with!

Worked example 1 (already set up)

Solve: 2x + 3y = 12 and −2x + y = 4
 2x + 3y = 12 +(−2x + y =  4)        4y = 16 → y = 4 Substitute y=4 into Eq 1: 2x + 12 = 12 → x = 0 Answer: (0, 4)

Worked example 2 (need to multiply first)

Solve: 3x + 2y = 8 and x − y = 1
Multiply Eq 2 by 2: 2x − 2y = 2  3x + 2y = 8 +(2x − 2y = 2)  5x       = 10 → x = 2 Substitute x=2 into Eq 2: 2 − y = 1 → y = 1 Answer: (2, 1)
Mr. Behling says: Don't let it freak you out — it's just two equations working together. Pick the method that looks easiest for the problem in front of you.
