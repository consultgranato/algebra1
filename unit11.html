11.1Zero & Negative Exponents ▼

Connect to prior knowledge: You already know that 23 = 2 × 2 × 2 = 8. That's repeated multiplication. Now we extend this idea to zero exponents and negative exponents.

Base, Exponent, Power
In the expression bn:
• b is the base — the number being multiplied
• n is the exponent — how many times to multiply
• The whole thing (bn) is a power

Example: 53 = 5 × 5 × 5 = 125  (base=5, exponent=3, power=125)
Zero Exponent Rule
Anything raised to the zero power equals 1.
b0 = 1  (for any b ≠ 0)

Examples: 50 = 1  •  x0 = 1  •  (−3)0 = 1  •  1000 = 1
Watch the parentheses!
(−3)0 = 1  ← the negative is inside, so the whole thing is raised to 0.
−30 = −(30) = −1  ← only 3 is raised to 0, then we negate.

Parentheses matter!
Negative Exponent Rule
A negative exponent means reciprocal (flip it):
a−n = 1 / an

• Negative exponent on top of a fraction: move it to the bottom, make the exponent positive.
• Negative exponent on bottom of a fraction: move it to the top, make the exponent positive.

Examples:
x−3 = 1/x3
1/x−4 = x4
2−3 = 1/23 = 1/8

Worked examples

Example 1 — Simplify: 4−2
4−2 = 1/4² = 1/16
Example 2 — Simplify: x−1 · y²
x−1 = 1/x = y² / x
Example 3 — Simplify: 3 / x−2
x−2 is on the bottom with a negative exponent Move x² to the top (exponent becomes positive) = 3x²
Joke: Mr. Behling: “I love all of algebra, but I have to be honest. Exponents are number 40 in my heart.” (40 = 1, so they're #1!)
11.2Multiplying & Dividing Powers ▼

Connect to prior knowledge: When you multiply or divide two powers with the same base, there are shortcuts so you don't have to expand everything.

Multiplying Powers — Same Base
am × an = am+n
When multiplying powers with the same base, ADD the exponents.

Why? x3 · x5 = (x·x·x)(x·x·x·x·x) = x8  ✓
Dividing Powers — Same Base
am / an = am−n
When dividing powers with the same base, SUBTRACT the exponents.

Top-heavy-bottom-heavy approach: bring the larger exponent to its side and subtract.

Example: x7 / x3 = x7−3 = x4
Coefficients are separate!
Handle coefficients (numbers out front) separately from the variable exponents.
(2x3)(5x2) = (2×5)(x3+2) = 10x5

Worked examples

Multiply: x3 · x5
Same base (x) → ADD exponents = x3+5 = x8
Multiply: (2x3)(5x²)
Coefficients: 2 × 5 = 10 Variables: x³ · x² = x3+2 = x5 = 10x5
Multiply: (4x²)(3x4)
Coefficients: 4 × 3 = 12 Variables: x² · x⁴ = x2+4 = x6 = 12x6
Divide: x9 / x3
Same base (x) → SUBTRACT exponents = x9−3 = x6
Divide: 12x7 / 4x2
Coefficients: 12 ÷ 4 = 3 Variables: x7−2 = x5 = 3x5
Divide: x5 / x8
5 − 8 = −3 = x−3 = 1/x³ (apply negative exponent rule)
11.3Raising a Power to a Power ▼

Connect to prior knowledge: Now we look at what happens when an entire power is raised to another power — like (x3)4.

Power to a Power Rule
(am)n = am × n
When a power is raised to a power, MULTIPLY the exponents.

Why? (x3)4 means x3 multiplied 4 times = x3+3+3+3 = x12  ✓
Distribute to ALL factors!
When you raise a product inside parentheses to a power, the outer exponent applies to every factor inside — coefficient AND variable(s).

(2x²)³ = 2³ · (x²)³ = 8x6  ← don't forget to cube the 2!

Worked examples

Example 1 — (x²)5
Multiply the exponents: 2 × 5 = 10 = x10
Example 2 — (3x4)²
Apply exponent to each factor: 3² = 9   and   (x4)² = x4×2 = x8 = 9x8
Example 3 — (2x³y)³
Apply exponent to each factor: 2³ = 8   and   (x³)³ = x9   and   y³ = y³ = 8x9y³

11.1
Power vs. Base vs. Exponent
In bn: b is the base (what is multiplied), n is the exponent (how many times), and the whole expression bn is called the power.
Example
53: base = 5, exponent = 3
53 = 5×5×5 = 125 (the power)
11.1
Anything raised to the zero power…
Any nonzero base raised to the exponent 0 equals 1. b0 = 1. This applies to numbers, variables, and expressions.
Example
70 = 1
x0 = 1
(-3)0 = 1
BUT: -30 = -1 (careful!)
11.1
Negative exponents on top
A negative exponent on the numerator (top) means move the factor to the denominator (bottom) and make the exponent positive.
Example
x-3 = 1/x3
2-4 = 1/24 = 1/16
11.1
Negative exponents on the bottom
A negative exponent in the denominator (bottom) means move the factor to the numerator (top) and make the exponent positive.
Example
1/x-4 = x4
3/y-2 = 3y2
New in 11.2
11.2
Shortcut for multiplying powers
When multiplying two powers with the SAME base, keep the base and ADD the exponents. am × an = am+n.
Example
x4 · x6 = x10
(add: 4+6=10)
11.2
Shortcut for dividing powers
When dividing two powers with the SAME base, keep the base and SUBTRACT the exponents. am / an = am-n.
Example
x9 / x3 = x6
(subtract: 9-3=6)
11.2
Top-heavy-bottom-heavy approach
When dividing powers, determine which side has the larger exponent. That side “wins.” Place the difference of the exponents on the winning side. Example: x3/x8 → bottom is heavier (8>3), so result is 1/x5.
New in 11.3
11.3
Shortcut for raising a power to a power
When a power is raised to another power, keep the base and MULTIPLY the exponents. (am)n = am×n. Apply to ALL factors inside parentheses.
Example
(x3)4 = x12 (multiply: 3×4)
(2x²)³ = 8x6
