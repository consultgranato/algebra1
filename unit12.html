12.1
Linear pattern
A pattern where you add or subtract the same constant each step. Forms a straight line.
Visual example
2, 5, 8, 11 → +3 each time
Constant difference → linear
12.1
Exponential pattern
A pattern where you multiply or divide by the same number (common ratio) each step. Forms a curve.
Visual example
2, 6, 18, 54 → ×3 each time
Common ratio → exponential
12.1
Standard equation for exp. functions
y = abx, where a is the y-intercept (starting value) and b is the common ratio (multiplier).
Visual example
y = abx
y = 3(2x)
a = 3, b = 2
12.1
The “a” in y = abx
The y-intercept and starting value of the exponential function. When x = 0, y = a (since b0 = 1).
Visual example
y = 5(3x)
When x=0: y = 5(1) = 5
a = 5 = y-intercept
12.1
The “b” in y = abx
The common ratio or multiplier. Divide any y-value by the previous one to find b. Determines growth or decay.
Visual example
y values: 4, 8, 16, 32
84 = 2 → b = 2
12.1
Exponential growth
When b > 1, the function grows — the y-values get larger and larger as x increases.
Visual example
y = 2(3x)
b = 3 > 1 → GROWTH
Curve rises steeply to the right
12.1
Average rate of change
The slope between two points on a curve: (f(x2)−f(x1))/(x2−x1). Unlike linear functions, this is NOT constant for exponential functions.
Visual example
f(2)=9, f(0)=1
Avg rate = (9−1)/(2−0) = 4
12.1
Exponential decay
When 0 < b < 1, the function decays — y-values shrink toward zero as x increases (but never reach zero).
Visual example
y = 8(0.5x)
b = 0.5, 0 < b < 1 → DECAY
Curve falls toward x-axis
New in 12.3
12.3
Examples of exponential growth
Population growth, bacterial reproduction, investment compound interest, spread of a virus (like COVID).
Visual example
Population → grows each year
Bacteria → doubles every hour
Investment → earns interest on interest
12.3
Examples of exponential decay
Radioactive decay, car depreciation (losing value), cooling of a hot liquid.
Visual example
Radioactive → half-life shrinks
Car → loses 15% value/year
Coffee → cools toward room temp
12.3
Exponential growth/decay formula
Growth: A = A₀(1+r)t — Decay: A = A₀(1−r)t. Where A₀ = initial amount, r = rate as a decimal, t = time.
Visual example
A = A₀(1+r)t or A = A₀(1−r)t
$1000 at 5% for 3 yrs:
1000(1.05)3 ≈ $1157.63
