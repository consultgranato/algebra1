12.1Exponential Functions (Part 1) ▼

Connect to prior knowledge: You know linear patterns — add or subtract the same amount each time. Exponential patterns are different: you multiply or divide by the same number (the common ratio) each time.

Two types of patterns
Linear: Add or subtract a constant each step (y goes up/down by the same amount)
  Example: 2, 5, 8, 11, 14 → +3 each time

Exponential: Multiply or divide by a constant each step (y grows/shrinks by a factor)
  Example: 2, 6, 18, 54 → ×3 each time
Standard form of an exponential function
y = abx

a = y-intercept (the starting value, where x = 0)
b = the common ratio (multiplier)
  • If b > 1 → GROWTH (curve goes up)
  • If 0 < b < 1 → DECAY (curve goes down toward zero)

Key attributes

Domain, Range, and y-intercept
y-intercept: at x = 0, y = a (plug in zero for x: b0 = 1, so y = a)
Domain: all real numbers
Range: y > 0 (for positive a; the curve never touches the x-axis)

Average Rate of Change

Formula
Average Rate of Change = (f(x₂) − f(x₁)) / (x₂ − x₁)
This is “rise over run” for curves — unlike lines, it changes at every interval!

Worked example — y = 2(3x)

Evaluate y = 2(3^x) for x = 0, 1, 2, 3
x = 0: y = 2(3&sup0;) = 2(1) = 2 x = 1: y = 2(3¹) = 2(3) = 6 x = 2: y = 2(3²) = 2(9) = 18 x = 3: y = 2(3³) = 2(27) = 54 b = 3 > 1 → GROWTH. y-intercept = 2. Domain: all reals. Range: y > 0.
Joke: The population of Ireland is exponential. Every year it's Dublin. (Doubling.)
12.2Exponential Functions (Part 2) ▼

Connect to prior knowledge: In 12.1 you evaluated functions. Now we work backwards — given a graph or table, determine the equation y = abx. Mr. Behling loves “backwards” problems!

How to find the equation from a table or graph
Step 1 — Find a: Read the y-intercept (where x = 0). That is your “a” value.
Step 2 — Find b: Divide any y-value by the previous y-value (consecutive values). That is your common ratio “b”.
Step 3 — Write it: y = abx
Growth or decay? If b > 1 → growth; if 0 < b < 1 → decay.

Worked example 1 — from a table

Table: x = 0, 1, 2, 3 and y = 5, 10, 20, 40
Step 1: x=0 gives y=5 → a = 5 Step 2: ratio = 10/5 = 2 → b = 2 Equation: y = 5(2^x) — GROWTH (b=2>1)

Worked example 2 — from a graph

Graph shows y-intercept at 8 and the curve is decreasing, halving each time
a = 8 (y-intercept from graph) b = 1/2 (halving each step) Equation: y = 8(1/2)^x — DECAY (0 < b < 1)
12.3Exponential Function Word Problems ▼

Connect to prior knowledge: Real-world situations often follow exponential patterns. You can now write and evaluate exponential growth and exponential decay models from word problems.

Formulas
Growth: A = A₀(1 + r)t
Decay: A = A₀(1 − r)t

A₀ = initial amount, r = rate (as a decimal), t = time
Real-world examples
Growth: Population, bacteria, investments, COVID spread
Decay: Radioactive decay, car depreciation, cooling

Worked example 1 — population growth

100 frogs, 22% annual growth — after 5 years
A = 100(1 + 0.22)^5 = 100(1.22)^5 = 100(2.703) ≈ 270 frogs

Worked example 2 — population decline

1200 birds, 5% decline, 2019–2029 (10 years)
A = 1200(1 − 0.05)^10 = 1200(0.95)^10 = 1200(0.5987) ≈ 718 birds

Worked example 3 — depreciation

Car worth $30,000 in 2026, depreciates 15%/year — value in 2030 (4 years later)
A = 30000(1 − 0.15)^4 = 30000(0.85)^4 = 30000(0.5220) ≈ $15,660
Joke: How do you know your dentist was good at algebra? She tells you that more candy leads to exponential decay.
12.4Solving Exponential Equations ▼

Connect to prior knowledge: When two exponential expressions are equal AND have the same base, their exponents must be equal. This is the key rule for solving exponential equations.

Key concept
If bm = bn, then m = n. (Same base, equal → exponents are equal.)

Method 1 (same base already): Set exponents equal, solve.
Method 2 (convert to same base): Rewrite using powers (e.g., 4 = 2², 8 = 2³, 9 = 3², 27 = 3³).
Method 3 (can't match bases): Use a graphing calculator — graph both sides, find intersection.
Powers to know cold
Powers of 2: 2, 4, 8, 16, 32, 64, 128, 256
Powers of 3: 3, 9, 27, 81
Powers of 4: 4, 16, 64
Powers of 5: 5, 25, 125

Worked examples

Example 1: Solve 2^x = 16
2^x = 16 = 2^4 Same base! Set exponents equal: x = 4
Example 2: Solve 3^x = 27
3^x = 27 = 3^3 Same base! x = 3
Example 3: Solve 4^x = 8 (different bases → convert to base 2)
4^x = 8 → (2^2)^x = 2^3 2^(2x) = 2^3 Same base! 2x = 3 → x = 3/2

Standard equation for exp. functions
y = ab^x, where a is the y-intercept (starting value) and b is the common ratio (multiplier).
Visual example
y = ab^x

12.1
The “a” in y = ab^x
The y-intercept and starting value of the exponential function. When x = 0, y = a (since b^0 = 1).
Visual example
y = 5(3^x)
When x=0: y = 5(1) = 5
a = 5 = y-intercept
12.1
The “b” in y = ab^x
The common ratio or multiplier. Divide any y-value by the previous one to find b. Determines growth or decay.
Visual example
y values: 4, 8, 16, 32
8/4 = 2 → b = 2
12.1
Exponential growth
When b > 1, the function grows — the y-values get larger and larger as x increases.
Visual example
y = 2(3^x)
b = 3 > 1 → GROWTH
Curve rises steeply to the right
12.1
Average rate of change
The slope between two points on a curve: (f(x2)−f(x1))/(x2−x1). Unlike linear functions, this is NOT constant for exponential functions.
Visual example
f(2)=9, f(0)=1
Avg rate = (9−1)/(2−0) = 4
12.1
Exponential decay
When 0 < b < 1, the function decays — y-values shrink toward zero as x increases (but never reach zero).
Visual example
y = 8(0.5^x)
b = 0.5, 0 < b < 1 → DECAY
Curve falls toward x-axis

Exponential growth/decay formula
Growth: A = A₀(1+r)^t — Decay: A = A₀(1−r)^t. Where A₀ = initial amount, r = rate as a decimal, t = time.
Visual example
A = A₀(1+r)^t or A = A₀(1−r)^t
$1000 at 5% for 3 yrs:
1000(1.05)^3 ≈ $1157.63
