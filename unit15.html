15.1Introduction to Quadratic Functions ▼

Connect to prior knowledge: Linear functions make straight lines. Quadratic functions make U-shaped curves called parabolas. The parent function is y = x².

Standard Form of a Quadratic
y = ax² + bx + c

a > 0: parabola opens UP (makes a U)
a < 0: parabola opens DOWN (makes an upside-down U)
|a| > 1: narrower than parent
|a| < 1: wider than parent
y-intercept: always the value of c (set x = 0)
Parent Function: y = x²
x −3 −2 −1 0 1 2 3
y 9 4 1 0 1 4 9
Notice: symmetric around x = 0 (the axis of symmetry). The vertex is at (0, 0).
Direction of Opening
Worked example — identify attributes of y = 2x² − 3x + 5
Identify a, b, c: a = 2, b = −3, c = 5 Direction: a = 2 > 0 → opens UP (minimum vertex) Width: |a| = 2 > 1 → narrower than parent y = x² y-intercept: set x = 0 → y = 5 → point (0, 5)

15.2Vertex & Axis of Symmetry ▼

Connect to prior knowledge: Every parabola has a vertex (the highest or lowest point) and a axis of symmetry (a vertical line through the vertex).

Finding Axis of Symmetry and Vertex
Axis of symmetry: x = −b / (2a)
Vertex: plug AOS x-value back into the equation to find y

If a > 0: vertex is the minimum
If a < 0: vertex is the maximum

Worked example — y = x² + 4x + 3

Worked example
a = 1, b = 4, c = 3 AOS: x = −4 / (2×1) = −4/2 = −2 Vertex y: y = (−2)² + 4(−2) + 3 = 4 − 8 + 3 = −1 Vertex: (−2, −1). Minimum (a > 0) ✓

15.3Graphing in Standard Form ▼

Connect to prior knowledge: To graph a quadratic from standard form, you follow a systematic 5-step process. Use Desmos to check your work!

5-Step Graphing Process (Standard Form)
Step 1: Find axis of symmetry: x = −b / 2a
Step 2: Find vertex (plug AOS into equation)
Step 3: Find y-intercept (set x = 0 → y = c)
Step 4: Find symmetric points (reflect y-intercept)
Step 5: Draw smooth parabola through all points

Worked example — y = x² − 6x + 8

Worked example — all 5 steps
a=1, b=−6, c=8 Step 1 — AOS: x = −b/(2a) = −(−6)/(2×1) = 6/2 = 3 Step 2 — Vertex: y = 3²−6(3)+8 = 9−18+8 = −1 → vertex: (3, −1) ← minimum Step 3 — y-intercept: x=0 → y=8 → point (0, 8) Step 4 — Symmetric point: (0,8) is 3 units left of AOS → mirror: (6, 8) Step 5: Draw upward parabola through (0,8), (2,0), (3,−1), (4,0), (6,8) ✓
Key Points for y = x²−6x+8
x y = x²−6x+8 Point
0 8 (0, 8) ← y-int
2 0 (2, 0) ← x-int
3 −1 (3, −1) ← vertex
4 0 (4, 0) ← x-int
6 8 (6, 8) ← symmetric to (0,8)
15.4Graphing in Vertex Form ▼

Connect to prior knowledge: Vertex form makes graphing even easier because the vertex is visible right in the equation.

Vertex Form
y = a(x − h)² + k

Vertex: (h, k) — read directly from equation
h: horizontal shift (opposite sign from what you see!)
k: vertical shift
a: determines direction and width (same as standard form)
Step-by-step from vertex form
Step 1: Read (h, k) directly from y = a(x − h)² + k → that's your vertex
Step 2: Check a: positive = opens up, negative = opens down
Step 3: Plot the vertex (h, k)
Step 4: Go 1 unit left and right from the vertex; go up (or down) |a| units
Step 5: Draw the smooth parabola through all three points
Watch the sign of h!
y = (x − 3)² + 1 → vertex is (3, 1) NOT (−3, 1). The h value is whatever makes (x − h) = 0.

Worked example — y = 2(x − 3)² + 1

Worked example
a = 2, h = 3, k = 1 Vertex: (3, 1). Opens up. Narrow (a=2). Graph: start at (3,1). Go right 1 → up 2: (4, 3). Go left 1 → up 2: (2, 3).

Worked example — y = −(x + 2)² + 4

Worked example — y = −(x+2)² + 4
Rewrite: y = −1(x − (−2))² + 4 Step 1 — Read vertex: h = −2, k = 4 → Vertex: (−2, 4) Step 2 — Direction: a = −1 < 0 → opens DOWN (maximum) Step 3 — Points: 1 right → (−1, 3); 1 left → (−3, 3) Step 4 — Draw downward parabola through (−3,3), (−2,4), (−1,3) ✓

15.1
Quadratic Function
A function of the form y = ax² + bx + c where a ≠ 0. Produces a parabola.
Example
y = x² → parent function
y = 2x² − 3 → quadratic
15.1
Parabola
The U-shaped curve produced by a quadratic function. Can open up or down depending on the sign of a.
Example
a > 0: opens up (U shape)
a < 0: opens down (∩ shape)

The highest or lowest point of the parabola. If a > 0, it's the minimum. If a < 0, it's the maximum.
Example
y = x² + 4x + 3
Vertex: (−2, −1)

15.2
Axis of Symmetry
A vertical line that divides the parabola into two mirror-image halves. Formula: x = −b / (2a).
Example
y = x² + 4x + 3
AOS: x = −4/(2·1) = −2
Equation: x = −2
