7.1
Relation
Any pairing of input values (x) with output values (y). Every function is a relation, but not every relation is a function.
Example
{(1,2),(3,4),(5,6)}
This is a relation (and a function).
7.1
Mapping
A diagram that shows which x-values (left bubble) pair with which y-values (right bubble) using arrows.
Example
1 → 5
2 → 7
3 → 5  ← ok! (two x's can share a y)
7.1
Table
A representation of a relation using two columns: one for x-values (input) and one for y-values (output).
Example
x | y
1 | 3
2 | 5
3 | 7
7.1
Ordered pair
A pair of numbers (x, y) that gives the coordinates of a point. The first number is always the x (input), the second is y (output).
Example
(3, 7) → x = 3, y = 7
Always x first, then y.
7.1
Graph (as a representation)
Points or a curve plotted on the coordinate plane, showing all ordered pairs of a relation visually.
Example
A set of dots on the grid.
Connected line = continuous.
Separate dots = discrete.
7.1
Function
A relation where each x-value is paired with EXACTLY ONE y-value. No x repeats with a different y.
Example
{(1,2),(2,5),(3,8)} → Function
{(1,2),(1,5)} → NOT a function
7.1
Discrete graph
A graph made of individual, unconnected dots. The data has separate, countable values (no values between the dots).
Example
Number of siblings: can be 0,1,2,3 — not 1.5.
→ Discrete
7.1
Continuous graph
A graph that is an unbroken connected curve or line. Values between any two points are also included.
Example
Temperature over time — every decimal temperature is possible.
→ Continuous
New in 7.2
7.2
Domain
The set of ALL x-values (inputs) used in a relation. “D comes before R, just like X comes before Y.”
Example
{(2,5),(4,7),(-1,3)}
Domain = {-1, 2, 4}
7.2
Range
The set of ALL y-values (outputs) used in a relation. These are the results after you apply the function.
Example
{(2,5),(4,7),(-1,3)}
Range = {3, 5, 7}
7.2
X-axis
The horizontal number line on the coordinate plane. Domain values live here (inputs).
Example
The line going left ←→ right.
Positive x is to the right.
7.2
Y-axis
The vertical number line on the coordinate plane. Range values live here (outputs).
Example
The line going up ↑↓ down.
Positive y is upward.
7.2
Origin
The point (0, 0) where the x-axis and y-axis cross. The starting center of the coordinate plane.
Example
The point exactly in the
middle of the grid: (0, 0)
7.2
Different names for the x-y plane
The coordinate plane is also called the Cartesian plane (after René Descartes), the xy-plane, or simply the coordinate plane. All mean the same thing: a grid with an x-axis and y-axis.
New in 7.3
7.3
Constant rate of change
When the ratio of change in y to change in x is always the same. This is the slope. Linear functions always have a constant rate of change.
Example
Table: y goes up by 3 every time x goes up by 1.
Rate of change = 3 (constant!) → linear
7.3
Linear function (graph)
A function whose graph is a straight line. The exponent on x is 1 (no x², no |x|, etc.).
Example
y = 2x + 1 → straight line ✓
y = x² → parabola, NOT linear
7.3
Linear function (table)
In a table, a linear function shows constant differences in the y-values when x increases by the same amount each row.
Example
x: 1,2,3 | y: 5,8,11
Differences: +3, +3 → linear ✓
7.3
Linear function (ordered pairs)
A set of ordered pairs is linear if the slope (rise over run) between any two pairs is always equal.
Example
(0,1),(1,3),(2,5): slope = 21 = 2 each time → linear ✓
New in 7.4
7.4
Function notation
Writing a function as f(x) instead of y. It names the function (f) and shows the input (x). f(x) = expression.
Example
y = 3x − 2 becomes f(x) = 3x − 2.
f(x) is NOT f times x!
7.4
How do you read “f(x)”?
Read as “f of x”. The letter before the parenthesis is the function name. The value inside is the input.
Example
f(3) = “f of 3” → plug in x = 3
g(0) = “g of zero” → plug in x = 0
7.4
“Forward” problems
Given an x-value, find f(x). Substitute the given number in for x and simplify. These go input → output.
Example
f(x) = 4x − 1. Find f(3):
f(3) = 4(3) − 1 = 11
7.4
“Backward” problems
Given the output f(x) = some value, find the input x. Set the expression equal to the given value and solve like an equation.
Example
f(x) = 4x − 1. If f(x) = 11, find x:
4x − 1 = 11 → 4x = 12 → x = 3
