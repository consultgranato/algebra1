16.1
Square Root Method
A method for solving quadratics with no x term. Isolate x2, take ±√ of both sides.
Example
x2 = 49
x = ±7
x = 7 or x = -7
16.1
± Symbol
Plus-or-minus. Indicates two solutions: one using + and one using -. Every positive number has two square roots.
Example
x = ±5 means
x = 5 OR x = -5
New in 16.2
16.2
Zero Product Property
If A × B = 0, then A = 0 OR B = 0. Allows you to split a factored equation into two simpler equations.
Example
(x-3)(x+5) = 0
x-3=0 → x=3
x+5=0 → x=-5
New in 16.3
16.3
Quadratic Formula
x = (−b ± √(b2−4ac)) / (2a). Works for ANY quadratic ax2+bx+c=0, including ones that can't be factored.
Visual example
x2−5x+6=0
a=1, b=−5, c=6
x = (5±√(25−24))/2
   = (5±1)/2
x=3 or x=2
New in 16.4
16.4
Discriminant
The expression b²-4ac. Determines the number of real solutions without fully solving. D>0: two solutions; D=0: one; D<0: none.
Example
x2-6x+9=0
D = 36-36 = 0
One real solution
