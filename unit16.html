16.1Solving by Square Roots ▼

Connect to prior knowledge: When a quadratic has no x term (just x² and a constant), you can solve it directly by taking square roots. Remember: every positive number has TWO square roots!

Square Root Method
Works when: no x term (b = 0 in ax² + bx + c = 0)

Steps: (1) Get x² alone   (2) Take ±√ of both sides   (3) Simplify

Always write ± — there are two solutions unless the right side is 0!

Worked examples

Example 1 — x² = 25
x = ±√25 = ±5 x = 5   or   x = −5 ✓
Example 2 — 3x² − 12 = 0
3x² = 12 x² = 4 x = ±2 ✓
Example 3 — x² + 16 = 0
x² = −16 → no real solution (√ of negative)
Joke: Why did the quadratic equation feel so lonely? Because it didn't know where its roots were.
16.2Solving by Factoring ▼

Connect to prior knowledge: From Unit 14 you learned to factor polynomials. Now you use factoring to solve equations by applying the Zero Product Property.

Zero Product Property (ZPP)
If A × B = 0, then A = 0 OR B = 0

Steps: (1) Move everything to one side (= 0)   (2) Factor   (3) Set each factor = 0   (4) Solve each

Worked examples

Example 1 — x² + 5x + 6 = 0
(x + 2)(x + 3) = 0 x + 2 = 0 → x = −2 x + 3 = 0 → x = −3 ✓
Example 2 — x² − 5x = 0
x(x − 5) = 0 x = 0   or   x = 5 ✓
Don't divide by x!
If x² = 5x, don't divide both sides by x — you'll lose a solution. Move everything to one side and factor.
16.3The Quadratic Formula ▼

Connect to prior knowledge: The quadratic formula works for ANY quadratic equation — even ones that can't be factored easily. It comes from completing the square on the general form.

The Quadratic Formula
For ax² + bx + c = 0:
x = (−b ± √(b² − 4ac)) / (2a)

Steps: (1) Write in standard form ax²+bx+c=0   (2) Identify a, b, c   (3) Plug in   (4) Simplify both solutions

Worked example — x² − 5x + 6 = 0

Worked example
a=1, b=−5, c=6 x = (5 ± √(25 − 24)) / 2 = (5 ± √1) / 2 = (5 ± 1) / 2 x = (5+1)/2 = 3    or    x = (5−1)/2 = 2 ✓

Worked example — 2x² + 3x − 2 = 0

Worked example
a=2, b=3, c=−2 x = (−3 ± √(9 + 16)) / 4 = (−3 ± √25) / 4 = (−3 ± 5) / 4 x = 2/4 = 1/2    or    x = −8/4 = −2 ✓
Joke: I tried to factor x²−4 but it fell apart into (x+2)(x−2).
16.4Discriminant & Types of Solutions ▼

Connect to prior knowledge: The expression under the radical in the quadratic formula is called the discriminant. It tells you how many real solutions exist WITHOUT solving the full equation.

Discriminant = b² − 4ac
D > 0: TWO distinct real solutions (parabola crosses x-axis twice)
D = 0: ONE real solution (parabola touches x-axis at vertex)
D < 0: NO real solutions (parabola doesn't cross x-axis)

Worked examples

Example 1 — x² − 5x + 4
D = (−5)² − 4(1)(4) = 25 − 16 = 9 D = 9 > 0 → two real solutions ✓
Example 2 — x² − 4x + 4
D = 16 − 16 = 0 D = 0 → one real solution (double root) ✓
Example 3 — x² + 2x + 5
D = 4 − 20 = −16 D = −16 < 0 → no real solutions ✓
