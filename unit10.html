10.1
Verbal Model
A word description of the relationships in a problem before writing it as an equation.
Example
"Three times a number plus five equals 20"
→ verbal model for 3x + 5 = 20
10.1
Variable Definition
Clearly stating what the variable represents. Always write "Let x = ..." before setting up an equation.
Example
Let x = the number of hours worked
Not just "let x = hours" — be specific!
10.1
Equation Setup
Translating the verbal model into an algebraic equation that can be solved.
Example
"5 less than twice a number is 11"
→ 2x − 5 = 11
New in 10.3
10.3
Perimeter Formula
Perimeter = sum of all sides of a polygon. For a rectangle: P = 2l + 2w. Geometry word problems often give the perimeter and a relationship between sides.
Example
Rectangle: length is 3 more than width
P = 26 → 2(w+3)+2w=26 → w=5
10.3
Angle Sum
The angles of a triangle always add to 180°. The angles of a quadrilateral add to 360°. Use this as the equation when angles are given as expressions.
Example
Triangle: angles are x, 2x, 3x
x+2x+3x=180 → 6x=180 → x=30°
New in 10.4
10.4
Flat Fee + Rate Model (b + mx = y)
Many real-world problems have a fixed starting cost (b) plus a per-unit charge (m) times quantity (x). Total cost y = b + mx. Given any two of these, solve for the third.
Example
Plumber: $75 flat + $50/hr
y = 75 + 50x
For 3 hrs: y = 75+150 = $225
New in 10.5
10.5
Two-Variable System
When two unknowns are related by two separate conditions, you write a system of two equations. Solve by substitution or elimination.
Example
Adults $10, kids $6, 200 tickets, $1600 total
a+k=200 and 10a+6k=1600
New in 10.6
10.6
Rate
Speed or amount per unit time. In d = rt, rate is how fast something moves or happens.
Example
60 miles per hour
$15 per hour
3 gallons per minute
10.6
Distance Formula d = rt
Distance = Rate × Time. The fundamental relationship connecting motion variables.
Example
r=50 mph, t=3 hr
d = 50 × 3 = 150 miles
New in 10.7
10.7
Mixture Problem
A problem combining two different quantities or concentrations. Use weighted averages: amount₁·value₁ + amount₂·value₂ = total value.
Example
Coffee A: $5/lb, Coffee B: $8/lb
Mix 10 lbs for $6.50/lb avg
→ set up 5x + 8(10-x) = 65
